buildFX

A collection of lean build tools for your JavaScript workflow.

build.js

Speed up hacking on your JavaScript project.

From a list of source files it produces a single .js file that is optimized using the Google Closure Compiler.

Get started »

build.hb

A tool for streamlining Handlebars templates for your Ember.js project.

Uses the Handlebars compiler that ships with Ember to precompile templates, producing a single .js file that you can include in your Ember.js project.

Get started »

build.spec

Test your JavaScript code using Jasmine-based BDD. Build.spec creates your test runner and keeps it up to date.
